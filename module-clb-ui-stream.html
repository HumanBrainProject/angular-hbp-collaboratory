Member: clbActivity: clb-activity directive is displays an activity retrieved by the HBP Stream service in a common way.

It try to look up for a detailled description of the event and fallback to the summary if he cannot.

Member: clbFeed: clb-feed directive displays a feed of activity retrieved by the HBP Stream service. It handles scrolling and loading of activities. Each activity is rendered using the clb-activity directive.
