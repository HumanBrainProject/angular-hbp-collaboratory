Function get¶

Return a promise that will resolve once the result set first page is loaded.

The promise contains the instance of the result set as well.

get(res[, options][, options.nextUrlKey][, options.previousUrlKey][, options.resultKey][, options.countKey][, options.resultsFactory])¶
Arguments:
  • res (Object) – a HTTPResponse or a promise which resolve to a HTTPResponse
  • options (Object) – configuration
  • options.nextUrlKey (string) – name of (or dot notation path to) the attribute containing the URL to fetch next results
  • options.previousUrlKey (string) – name of (or dot notation path to) the attribute containing the URL to fetch previous results
  • options.resultKey (string) – name of (or dot notation path to) the attribute containing an array with all the results
  • options.countKey (string) – name of (or dot notation path to) the attribute containing the number of results returned
  • options.resultsFactory (function) – a function to which a new array of results is passed. The function can return undefined, a promise or an array as result.
Return ResultSet:
 

a new instance of ResultSet
