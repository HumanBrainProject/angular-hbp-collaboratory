Children¶

Description¶

The clbFormControlFocus Directive mark a form element as the one that should receive the focus first.

Examples¶

Give the focus to the search field¶
angular.module('exampleApp', ['clb-form']);

// HTML snippet:
// <form ng-app="exampleApp"><input type="search" clb-form-control-focus></form>
